# 字典树

# 简介

设字符集为 ，字典树（Trie）是一颗 叉树

即一个节点至多有 个孩子，每一个孩子都表示当前字符可达的下一个字符

字典树的叶子是一种记号节点，它不存储字符，它表示一个字符串到此结束，它的作用有点像 '\0'
根节点实际上也是不存储字符的，它也是一种记号节点，表示所有字符串的开始

# 板子

struct Trie {
    int cnt = 1;
    int node[N][26]{};  // N 是最大字符数量，即最大输入数 * 字符集
    bool mark[N]{};

    void add(const string& s) {
        int ptr = 1;
        for (auto& c : s) {
            if (node[ptr][c - 'a'] == 0)
                node[ptr][c - 'a'] = ++cnt;
            ptr = node[ptr][c - 'a'];
        }
        mark[ptr] = true;
    }

    bool exist(const string& s) {
        int ptr = 1;
        for (auto& c : s) {
            if (node[ptr][c - 'a'] == 0)
                return false;
            ptr = node[ptr][c - 'a'];
        }
        return mark[ptr];
    }
} trie;

# 作用

# 判定特定的字符串是否在集合中

给一个字符串集合 S，然后多个询问，每个询问是一串字符串，问它是否在 S 中

这是字典树的本职工作，它就是用来干这个的，它的查询速度与待查字符串的长度成线性关系

习题：P2580 (opens new window)、P8306 (opens new window)

# 0-1 字典树

给一个整数集合 S，然后多个询问，每个询问是一个数字 x，要求从 S 中选出一个数 y，使得 x ^ y 最大

0-1 字典树是字符集为 {0, 1} 的一颗字典树，它实际上是一颗二叉树。如果将整数的二进制序列视为一个字符串，那么就可以在集合 S 上建立一颗字典树

假设 ，此时如果有 当然是最好的，因为它们可以得到 ，如果很不巧集合 S 中没有 ，那若是有 也是好的

从上面的例子可以看出，y 应该尽可能地在高位上与 x 相异，实在没有选择时才与 x 相同

而如果在集合 S 上有一颗 0-1 字典树，那就可以有选择地从根走到叶子，使得高位尽可能地与 x 相异，这样就选出了一个与 x 异或起来最大的 y

习题：U109895 (opens new window)、P4551 (opens new window)

# 可删除特定的字符串的字典树

给一个数组，从中选出三个数 a, b, c，使得 (a + b) ^ c 最大

可删字典树实质上是在原来字典树的基础上，让各个节点再记录自己有几个孩子，如果一个节点有 0 个孩子，那就认为它不在字典树中（或者认为它是叶子节点 '\0'）

DETAILS

在这种理解下，根是没有孩子的（不是 0，而是“没有”），仔细想想，根的孩子数其实是无法被查询的。这也说明根在字典树中是一个十分特别的存在

另一种理解是，这个多加的状态是这个节点被经过的次数

struct Trie {
    int cnt = 1;
    int node[N][26][2]{};  // 多一个状态，0 存储原来的指针，1 存储孩子个数
    bool mark[N]{};
    
    void add(const string& s) {
        int ptr = 1;
        for (auto& c : s) {
            if (node[ptr][c - 'a'][0] == 0)
                node[ptr][c - 'a'][0] = ++cnt;
            node[ptr][c - 'a'][1]++;
            ptr = node[ptr][c - 'a'][0];
        }
        mark[ptr] = true;
    }
    
    void remove(const string& s) {
        int ptr = 1;
        for (auto& c : s) {
            node[ptr][c - 'a'][1]--;
            ptr = node[ptr][c - 'a'][0];
        }
    }
    
    bool exist(const string& s) {
        int ptr = 1;
        for (auto& c : s) {
            if (node[ptr][c - 'a'][0] == 0 or node[ptr][c - 'a'][1] == 0)
                return false;
            ptr = node[ptr][c - 'a'][0];
        }
        return mark[ptr];
    }
} trie;
