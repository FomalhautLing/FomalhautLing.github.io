# 树状数组

# 简介

树状数组（Binary Index Tree. BIT）是一棵节点从 1 到 n 编号的树。它可能有多个根节点，若它有唯一的根，则一定是 n 号节点，此时 n 一定是 2 的整数次幂

每个二进制数都有一个最低位的 1，对于 i 号节点，假设 i 的二进制为 ，那么它的孩子有：，其中 x 表示一位 0 或 1

其规律为：假设 j 是 i 的孩子，那么一定有 j + low_bit(j) = i

如果没有 8 号节点，那么 4、6、7 就成为这棵树的根（其实是森林）

树状数组实际上是一颗前缀树，以前缀和为例，图中的 4 号节点实际上存储了 2、3 号节点与它本身数据的和，8 号节点存储了 4、6、7 号节点与它本身数据的和

# 板子

struct BIT {
private:
    int n;  // [1, n]
    vector<int> base;
    vector<int> tree;

    static inline size_t low_bit(size_t x) { return x & -x; }
public:
    explicit BIT(int n) : n(n) {
        base.resize(n + 1, 0);
        tree.resize(n + 1, 0);
    };

    void add(size_t index, int value) {
        base[index] += value;
        for (size_t i = index; i <= n; i += low_bit(i)) {
            tree[i] += value;  // 这里可以是 + ^ 或 minmax
        }
    }

    int sum_m(size_t m) {  // 前 m 项和
        int ans = 0;
        for (size_t i = m; i >= 1; i -= low_bit(i)) {
            ans += tree[i];
        }
        return ans;
    }
};

# 作用

# 单点修改，区间查询

给一个序列，接着有两种操作：1）在某个位置加上 x；2）求某个区间的和

这是树状数组的本职工作，它就是干这个的

例题：P3374 (opens new window)、P2068 (opens new window)

# 区间修改，单点查询

给一个序列，接着有两种操作：1）在给某个区间所有数加上 x；2）回答某个位置上的值

这是树状数组很漂亮的变形，把树状数组当成差分数组用，它就能够维护区间修改和单点查询

例题：P5057 (opens new window)、P5057 (opens new window)、P4939 (opens new window)

# 维护区间最值

给一个序列，然后数个询问，询问给出 l, r，回答区间 [l, r] 的最值

树状数组可以维护区间最值，时间复杂度
它返回闭区间 [l, r] 的最大/最小值，这取决于传入的函数

struct BIT {
    ...
    template<typename Tp>
    Tp minmax_element(size_t l, size_t r, Tp(* minmax)(Tp, Tp)) {  // 传 max<int> 即可
        if (l == r) return base[l];
        if (r - low_bit(r) > l) {
            return minmax(tree[r], minmax_element(l, r - low_bit(r), minmax));
        }
        else return minmax(base[r], minmax_element(l, r - 1, minmax));
    }
}
