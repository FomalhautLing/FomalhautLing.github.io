# 字典树

Trie，即字典树，是一个非常强大的数据结构，可以高效处理字符串、异或最大值以及数集维护问题。而且常数非常小。

  • P2580 (opens new window)

板子题，直接建立字典树模拟即可

  • SP4033 (opens new window)

两个思路

  1. 第一种：跑两遍，顺序读入字典树，这样可以判断后者是否是前者的前缀；然后倒序输入字典树，判断前者是否是后者的前缀
  2. 第二种：在插入时，如果某串没有新建尾结点，那么它是一个前缀；或者它经过了某个尾节点，那么这个尾节点的字符串是它的前缀
  • U109895 (opens new window)

首先对给定的数组建立 01-字典树。然后对于每次询问，从二进制高位向低位查找，如果询问数 s 当前位是 1，那么字典树优先走 0，如果是 0，优先走 1，这样保证异或后各位尽可能是 1，值也就最大

对于上面的选择有个技巧：$0 \oplus 1 = 1，1 \oplus 1 = 0$，每次只要往 $bit \oplus 1$ 的位置走就行

  • U109897 (opens new window)

本题涉及字典树的删除，枚举 $s_i s_j$，每次从字典树中将它们删去，然后跑一遍 U109895，维护最大值，最后再把 $s_i s_j$ 添加回来

删除其实只要多给一个状态，表示这个节点被经过的次数就可

  • U109923 (opens new window)

字典树 + dp

区间不交问题可以枚举分界点

定义 $max_l[x]$ 表示区间 $[1, x]$ 内最大的异或值，$max_r[x]$ 表示区间 $[x, n]$ 内的最大异或值。那么答案就是 $O(N)$ 枚举分割点计算一遍

对于 $max_l[x]$，有 $max_l[x] = max(max_l[x - 1], \oplus_i^x(a_i))$。区间异或实际上等于前缀异或的异或，于是维护前缀异或，要求 $\oplus_i^x(a_i)$，只要查找能与当前的前缀异或计算得到最大值的前缀异或即可（即字典树存前缀异或）

注意提前把 0 插入字典树

signed main() {

    ios::sync_with_stdio(false);
    cin.tie(nullptr), cout.tie(nullptr);

    int n; cin >> n;
    int arr[n]; for (let i in arr) cin >> i;

    trie.reset();

    int max_l[n + 2], pre_xor = 0; max_l[0] = 0;
    trie.insert(0);
    for (int i = 1; i <= n; i++) {
        int x = arr[i - 1];
        pre_xor ^= x;
        max_l[i] = max(max_l[i - 1], trie.find(pre_xor) ^ pre_xor);
        trie.insert(pre_xor);
    }

    trie.reset();

    int max_r[n + 2], las_xor = 0; max_r[n + 1] = 0;
    trie.insert(0);
    for (int i = n; i >= 1; i--) {
        int x = arr[i - 1];
        las_xor ^= x;
        max_r[i] = max(max_r[i + 1], trie.find(las_xor) ^ las_xor);
        trie.insert(las_xor);
    }

    int ans = 0;
    for (int i = 1; i < n; i++) {
        ans = max(ans, max_l[i] + max_r[i + 1]);
    }
    cout << ans << '\n';

    return 0;
}

  • P4551 (opens new window)

任意两个节点 $<a, b>$ 路径上的异或和实质上等于 $<1, a> \oplus <1, b>$，故 dfs 处理出所有点到 1 的路径异或，然后字典树找两个数使得其异或最大即可

  • P8306 (opens new window)

先把要查询的字符串丢进字典树，得到它们在字典树中的标记。然后再挨个输入数据，如果某个字符串在插入字典树的过程中中遇到了标记，说明这个标记代表的字符串是现在输入的这个的前缀

# 总结

字典树一般用来处理对一系列字符串的前缀查询问题，也处理最大异或，或者与异或相关的运算问题。常数 N 不好确定

  • P2880 (opens new window)

树状数组维护区间最值

树状数组原来的 query 并不能算出区间最值，需要改写为

  • 若 $r - lowbit(r) > l$，那么可以以 $r - lowbit(r)$ 二分，又 $[r - lowbit(r), r]$ 的最值就是 $tree[r]$
  • 若 $r - lowbit(r) \le l$，那么可以从 r 分开，往下递归，期待满足上一个条件的情况发生
int find_max(int l, int r) {
    if (l == r) return base[l];
    if (r - low_bit(r) > l) return max(tree[r], find_max(l, r - low_bit(r)));
    else return max(base[r], find_max(l, r - 1));
}

# 总结

树状数组的题一般都是给多组询问，每个询问有两个状态，改或者查。原始的树状数组可以维护单点修改和区间查询，树状差分数组可以维护区间修改单点查询

# 差分约束

给出一组包含 $m$ 个不等式，有 $n$ 个未知数的形如：

$$ \begin{cases} x_{c_1}-x_{c'1}\leq y_1 \x{c_2}-x_{c'2} \leq y_2 \ \cdots\ x{c_m} - x_{c'_m}\leq y_m\end{cases}$$

的不等式组，求任意一组满足这个不等式组的解。

  • P5960 (opens new window)

板子题，无解 => 有负环 => spfa

$ x_a - x_b \le y \Rightarrow$ 从 b 向 a 连边

  • P1993 (opens new window)

  • 一些变形相关的结论：

    1. $ x_1 - x_2\ge y \Rightarrow x_2 - x_1 \le -y $
    2. $ x_1 - x_2 = y \Rightarrow x_1 - x_2 \le y \wedge x_2 - x_1 \le -y $
    3. $ x_1 - x_2 < y \Rightarrow x_1 - x_2 \le y - 1 $
    4. $ x_1 - x_2 > y \Rightarrow x_2 - x_1 \le -y - 1 $

  • SP116 (opens new window)

$$ x_b - x_{a - 1} \ge c \ x_i - x_{i - 1} \ge 0 \ x_{i - 1} - x_i \ge -1 $$

这是一个未知数个数为 $x_d$(d 为最大的端点)，方程个数为 $n + 2(d - 1)$ 的差分约束问题。注意此处应跑最长路。这里图天然联通，不需要超级源点
