# 打印自我的程序

直觉上来说，不借助文件读写就能打印自我的程序似乎有点不可思议，但它确实存在

这样的程序被称为 Quine，它的运行结果就是打印出自身的源代码

s = 's = %r\nprint(s %% s)'
print(s % s)
